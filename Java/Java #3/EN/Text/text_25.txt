A common example of user input in Java is reading numbers from the keyboard, processing them, and then displaying the result to the user. This process is typically done using the Scanner class, which allows developers to collect input in a simple and structured way. The first step is to import the Scanner class from the java.util package, which provides the necessary tools for handling input. After that, a Scanner object is created using System.in, which connects the program to the keyboard input stream.

Once the Scanner object is ready, the program prompts the user to enter data, such as numbers. For example, the program may ask the user to enter the first number, and then use the nextInt() method to read that value and store it in a variable. The same process is repeated for the second number. At this point, both inputs are stored in variables, making them ready for processing.

After collecting the input, the program performs a specific operation, such as calculating the sum of the two numbers. This is done by simply adding the variables together and storing the result in another variable. Finally, the program displays the result to the user using output statements like System.out.println(), allowing the user to see the outcome of their input.

This example demonstrates the full flow of user input handling in Java: importing the necessary class, creating a Scanner object, asking for input, reading and storing values, processing the data, and displaying the result. It highlights how Java programs can interact with users in a clear, logical, and user-friendly way, forming the foundation of many real-world applications.
