Variable initialization is the process of assigning a value to a variable either at the moment it is 
declared or before it is used in a program. In Java, every variable must first be declared with a specific 
data type and a name, which defines what kind of data it can store. After declaration, initialization gives 
the variable its first value. For example, declaring a variable as int 
age; only creates the variable, but it does not yet contain a 
meaningful value. When we write age = 20;, we assign an initial value, completing the initialization step.

In some cases, variables can be both declared and initialized in a 
single line, such as double price = 19.99;, which improves code readability and efficiency. 
Java also provides default values for certain types of variables, especially class-level variables 
(like instance and static variables), but local variables inside methods do not receive default values automatically. 
This means they must be explicitly initialized before being used; otherwise, the program will produce a compile-time error.

The general flow of working with variables is: first declare the variable, then assign it a value, and finally 
use it in the program. For instance, after initializing age, it can be used in statements like System.out.println(age); 
to display its value. Proper initialization is crucial because it ensures that variables hold predictable and meaningful 
data, preventing errors and making the program more reliable and understandable.
